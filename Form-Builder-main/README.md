# Form Builder

A full-stack form builder application that allows users to create, manage, and analyze forms with ease.

## Features

- Create and customize forms with drag-and-drop interface
- Multiple question types (text, number, select, etc.)
- View and manage form responses
- Responsive design for all devices

## Tech Stack

### Frontend
- React 18
- Vite
- Tailwind CSS
- React Hook Form

### Backend
- Node.js
- Express.js
- MongoDB

## Getting Started

### Prerequisites
- Node.js (v16+)
- MongoDB (local or Atlas)
- npm or yarn
